# 美股技术指标监测程序 (MACD & RSI)

这是一个用于监测美股（包括股票和ETF基金）主要技术指标MACD和RSI的程序，能够检测买入和卖出信号。

## 功能特性

- 实时获取美股数据（支持股票和ETF）
- 计算MACD（移动平均收敛发散）指标
- 计算RSI（相对强弱指数）指标
- 检测MACD和RSI的买入/卖出信号
- 显示超买/超卖状态
- 综合信号分析

## 技术指标说明

### MACD（移动平均收敛发散）
- **买入信号**: MACD线上穿信号线
- **卖出信号**: MACD线下穿信号线

### RSI（相对强弱指数）
- **买入信号**: RSI从下向上突破30（超卖区反弹）
- **卖出信号**: RSI从上向下跌破70（超买区回调）
- **超卖状态**: RSI < 30
- **超买状态**: RSI > 70

## 使用方法

1. 安装依赖包：
```bash
pip install yfinance pandas numpy
```

2. 运行程序：
```bash
python stock_indicator_monitor.py
```

3. 输入要监测的股票代码，例如：
   - AAPL (苹果)
   - TSLA (特斯拉)
   - QQQ (纳斯达克ETF)
   - SPY (标普500ETF)
   - 等等

4. 输入 'quit' 退出程序

## 输出信息

- 当前日期和收盘价
- MACD值和信号线值
- RSI值及市场状态（超买/超卖/正常）
- 最近的MACD和RSI买入/卖出信号日期
- 同时出现MACD和RSI信号的综合分析
